# Presets

The plugin ships with three Flat Config-ready presets.

## Available presets

- [`recommended`](./recommended.md)
- [`recommended-ci`](./recommended-ci.md)
- [`recommended-detailed`](./recommended-detailed.md)

## Which preset should you choose?

- Choose **recommended** for normal local development.
- Choose **recommended-ci** when the same config is used in CI and local workflows.
- Choose **recommended-detailed** when you want the richer completion summary enabled by default.
